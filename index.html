Docker Topology Live

Real-time Docker container topology scanner and browser visualiser.

# Install
pip install -e .
# Start server (sample data)
python app.py serve --sample
# Start server (live Docker)
python app.py serve
# Export topology
python app.py sample --output topology.json

Once the server is running, open http://127.0.0.1:8080/
